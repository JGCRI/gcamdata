# gcamdata
The increasing data requirements of complex models demand robust, reproducible, and transparent systems to track and prepare modelsâ€™ inputs. The **gcamdata** R package processes raw inputs to produce the hundreds of XML files needed by the [GCAM integrated human-earth systems model](https://www.geosci-model-dev-discuss.net/gmd-2018-214/) which is available on its [Github repository](https://github.com/JGCRI/gcam-core).

The package is documented in the [online manual](https://jgcri.github.io/gcamdata/index.html)
